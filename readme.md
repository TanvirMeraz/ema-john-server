# Ema-john-server

This project is made with NodeJS, ExpressJS, MongoDB.

## Tech Stack

NodeJS: "18.15.3",\
ExpressJS: "4.18.2",\
cors: "2.8.5",\
dotenv: "16.0.3",\
express: "4.18.2",\
jsonwebtoken: "9.0.0",\
mongodb: "5.1.0"

## Run Locally

Clone the project

```bash
  git clone https://github.com/ChaionSaha/ema-john-server
```

Go to the project directory

```bash
  cd ema-john-server
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn start
```
